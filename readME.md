## About The Project

This repository is merely intended to document some of the challenges I've had on the course of the first semester of 2022 in the class of "Fundamentals of Programming", part of the undergratuate program 'Technology of Computing Systems' at Universidade Federal Fluminense.

### Built With

- [Python](https://www.python.org/)

<!-- USAGE EXAMPLES -->

## Usage

In each of the directories, there are python files with the solutions for the questions/challenges listed in the pdf file.

The files can be run using the python/python3 command plus the name of the file: "python q1.py" or "python3 q1.py".

Alternatively, it can be run within any IDE which has a terminal for running python code and inputting data.

PS. All of the descriptions of the tasks are in portuguese.
